# Usage in Twig Instructions

<br>

### Use `asset` function

```twig
{{ asset(your_entity.your_field_name.path) }}
# Or
<img class="img-responsive" src="{{ asset(your_entity.your_field_name.path) }}" alt="{{ your_entity.your_field_name }}">
```
